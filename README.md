### mn-ssr
mini react ssr framework (服务端使用 jsdom)


### 说明
index.html 模板 + jsx 固定入口 的 ssr 方案


### 命令
```shell
npm run dev # 开发模式
npm run build # 编译 SSR 代码和 SPA 代码
npm run build:client #只编译 SPA 代码
npm run build:server # 至编译 SSR 代码

```
